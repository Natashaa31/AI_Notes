Chapter 2 – Intelligent Agents

Agent

An agent is anything that perceives its environment through sensors and acts upon the environment through actuators.

Examples of Agents

  • Human agent
  • Robotic agent
  • Software agent

Percept

A percept is the agent’s perceptual input at a given instant.

Agent Function

An agent function maps a percept sequence to an action.

Agent Program

The implementation of an agent function running on a physical architecture.

Characteristics of Intelligent Agents

  • Sensing the environment
  • Acting upon the environment
  • Autonomous behavior
  • Rational decision making
